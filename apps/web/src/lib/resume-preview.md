## Color Scheme

- **Primary Blue (#002ad2)**: Headings, section titles, links
- **Dark Gray (#1a202c)**: Main body text
- **Medium Gray (#4a5568)**: Secondary text, dates
- **Light Gray (#e2e8f0)**: Table backgrounds, accents

## Typography

- **Font**: Calibri (Professional, ATS-friendly)
- **Name**: 36pt, Bold
- **Section Headers**: 28pt, Bold
- **Subsection Headers**: 24pt, Bold
- **Body Text**: 22pt, Regular
- **Small Text**: 20pt, Regular

## Spacing

- **Page Margins**: 0.75 inches all sides
- **Section Spacing**: 300-400 twips between sections
- **Paragraph Spacing**: 150-200 twips between items
- **Line Spacing**: Default (single)
